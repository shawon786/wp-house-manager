== Description ==

Manage your renters online now! Use this plugin to manage all your renters including there advances, monthly rents, dues, bills etc. Forget about writing in the books! Keep all the data online and notify them via email!

Allows you to change the default titles, colors according to your choice!

[youtube https://youtu.be/UKsUE_l-0IM]

Core Features (FREE):
More than a Plugin – A Real House Management System!

* MANAGE YOUR FLATS
You can add as much flats as you want. Even you can add flats of multiple buildings! Just use slug like- b1, b2 (b1 for building one, b2 for building two) on the front of flat name! Simple!

You will be able to see total flats, rented flats and available flats for to-let at a glance on the OVERVIEW page!

* MANAGE YOUR RENTERS
Add renters' detailed information for security purpose. Now there is no worries about data loss! All the data of the renters like National ID No, Passport No, Driving License No. etc will be stored online!

You can even store copies of these documents online by using our DOCUMENT MANAGER pro extension (coming soon). By using this PRO add-on you will be able to upload any types of files like- copy of NID, Driving License, Passport, agreement and so on!

* INVOICE / BILL MANAGEMENT SYSTEM
With this amazing feature you will be able to manage all bills from the first day of adding a renter. Just make invoice/bills for- Advance, House rent, Current bill or any other bills. And mark it as Complete (paid) once the bill is paid!

* EMAIL INVOICES/BILLS as PDF.
You can email the bills/invoices to the renters as PDF! This will help you to inform your renters about the dues or bills.

* EXTENSION (PRO)-[UTILITY MANAGER](https://voidcoders.com/product/house-manager-utility/).
Allows you to customize the looks and color as well. Change Brand Name, Logo, Titles, Sidebar color and so on according to your choice!

== Installation ==
### Automatic Install From WordPress Dashboard

1. Login to your the admin panel
2. Navigate to Plugins -> Add New
3. Search **House Manager**
4. Click install and activate respectively.

### Manual Install From WordPress Dashboard

If your server is not connected to the Internet, then you can use this method-

1. Download the plugin by clicking on the blue button above. A ZIP file will be downloaded.
2. Login to your site’s admin panel and navigate to Plugins -> Add New -> Upload.
3. Click choose file, select the plugin file and click install

###Install Using FTP

If you are unable to use any of the methods due to internet connectivity and file permission issues, then you can use this method-

1. Download the plugin by clicking on the blue button above.A ZIP file will be downloaded.
2. Unzip the file.
3. Launch your favorite FTP client. Such as FileZilla, FireFTP, CyberDuck etc. If you are a more advanced user, then you can use SSH too.
4. Upload the folder to wp-content/plugins/
5. Log in to your WordPress dashboard.
6. Navigate to Plugins -> Installed
7. Activate the plugin

* Once installed and activated, you will find a new option under "your site name" (at the top left, beside WordPress logo). So just click on "House Manager" to see House Manager at the frontend.
You can also hit "www.yourdoamin.com/house-manager" as well.
